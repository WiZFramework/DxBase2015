Simple05．３Ｄオブジェクトの頂点色テクスチャ描画


テクスチャと頂点カラーの掛け合わせ

　DxBase2015Simple/Simple05のソリューションを開いて、リビルド、実行しますと以下の画面が現れます。

 

図Simple05a

 

　今項では、頂点に色を付けてグラデーションを表現し、テクスチャも実装しています。
　画面のように、独特の効果が出るのがわかります。
　今項は、前項と前々項のサンプルの合体です。ですから、頂点がVertexPositionNormalColorTexture型になったということに注意すれば、他は前項、前々項のサンプルの応用です。
　一つ注目点とすれば、ピクセルシェーダで、最終的な色の決定に 
    //入力色とテクスチャ色とライト色を合成
    return (input.Diffuse + TextureColor) * LightColor;
　と、頂点色とテクスチャを加算した後、ライティングを掛け算しています。
　これを、 
    return input.Diffuse * TextureColor * LightColor;
　とすると、以下のようになり、ちょっと不気味な感じを演出できます。

 

図Simple05b

 

　今回は使用してませんが、コンスタントバッファに色を渡すこともできます。すると、頂点色、テクスチャ、ライティング、そしてコンスタントバッファと、様々な係数からピクセル色を決定できる形になります。
　このように、ピクセルシェーダの出力は、最終的なピクセル色の決定なので、それまで計算等で導き出した結果をどのような形で料理しても問題ありません。ぜひ自分なりのピクセル色を演出してみましょう。
